Every prompt in this collection is engineered based on these core principles:

-   **Precise Role-Playing:** Assigning the AI a specific, expert persona to enhance the depth and quality of its output.
-   **Crystal-Clear Instructions:** Eliminating ambiguity to prevent misinterpretation by the model.
-   **Forced Output Structure:** Exerting full control over the output format for downstream use (e.g., software imports, data processing).
-   **Self-Correction Mechanisms:** Integrating checklists and statistical analyses to compel the model to review and validate its own work.
